If you're developing an EHR and are looking to integrate with Nabla, you may pick one of the strategies below.

[Get in touch](https://www.nabla.com/copilot-contact/) if you wish to discuss your use case in more detail.

| Method                                            | For                                              | Pros                                                                                                                                                                                        | How                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| ------------------------------------------------- | ------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Front-end integration                             | Web-based EHRs                                   | Simple and straightforward flow, great for proof-of-concept and simple integrations.<br /><br />No need to set up the Nabla Core API.<br /><br />Ability to leverage the existing Nabla UI. | Using the Nabla Assistant [Chrome extension](https://chrome.google.com/webstore/detail/nabla-copilot/gdhbaoemgglcgmkidhnhcellgnehaeol).<br /><br />On your EHR domain, you may request the "Copy" button from the extension to be replaced with an "Export to `<YOUR_EHR>`" button. In that case, Nabla Assistant will export the generated note and structured information via JavaScript.<br /><br />The EHR web app should listen to `window` events to receive the required information when the button is clicked, and fill the right fields in the EHR (see example below). |
| Back-end integration (through the Nabla Core API) | All EHRs including ones available on mobile apps | For deep UI integration.<br /><br />Highly customizable.                                                                                                                                    | Server-side integration using Nabla APIs.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
