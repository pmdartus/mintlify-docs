---
title: "Dictation commands"
description: "Reference of dictation commands interpreted by the dictate endpoints."
icon: "microphone"
---

The following commands are interpreted and transformed into characters by the dictate endpoints:

| Action                  | Character | Keywords/Commands                                                                                                                        |
| ----------------------- | --------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| Add a new line          | `\n`      | next line <br/> line break <br/> new line                                                                                                |
| Add a new paragraph     | `\n\n`    | next paragraph <br/> new paragraph <br/> paragraph                                                                                       |
| Add a period            | `.`       | dot <br/> period <br/> full stop                                                                                                         |
| Add a comma             | `,`       | comma                                                                                                                                    |
| Add a question mark     | `?`       | question mark                                                                                                                            |
| Add an exclamation mark | `!`       | exclamation mark <br/> exclamation point                                                                                                 |
| Add an ellipsis         | `...`     | ellipsis                                                                                                                                 |
| Add a semicolon         | `;`       | semi-colon                                                                                                                               |
| Add a colon             | `:`       | colon                                                                                                                                    |
| Add quotes              | `"`       | quote <br/> double quotation mark <br/> double quote <br/> open quote <br/> close quote <br/> open double quote <br/> close double quote |
| Add single quotes       | `'`       | single quote <br/> open single quote <br/> close single quote <br/> single quotation mark                                                |
| Add open parentheses    | `(`       | left paren <br/> left parentheses <br/> open paren <br/> open parentheses <br/> open parenthesis                                         |
| Add close parentheses   | `)`       | close paren <br/> close parentheses <br/> close parenthesis <br/> right paren <br/> right parentheses <br/> right parenthesis            |
| Add open bracket        | `[`       | left bracket <br/> open bracket                                                                                                          |
| Add close bracket       | `]`       | right bracket <br/> close bracket                                                                                                        |
| Add a dash              | `-`       | dash <br/> hyphen <br/> minus                                                                                                            |
